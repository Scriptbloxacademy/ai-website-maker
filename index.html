Frequently asked questions

Everything you need to know about AI Website Maker

What is AI Website Maker?
AI Website Maker is a free tool that uses GPT-4o to build complete professional websites from a description. You describe what you want, and it plans the architecture, generates content, builds the HTML, and publishes it live on GitHub Pages — all in under 2 minutes.
How does it work?
Describe your website in plain English. Pick a template type and style. Our AI runs a multi-step pipeline: first it plans the site architecture, then generates professional content, then builds the HTML using hand-crafted templates. Finally, it creates a GitHub repository and publishes the site via GitHub Pages. You can then refine the site by telling the AI what to change.
Do I need coding experience?
No. Just describe what you want in plain English. The AI handles all the technical implementation — HTML, CSS, responsive design, animations. You don't need to write a single line of code.
What do I need a GitHub token for?
A GitHub classic token with the repo scope is required so the tool can create a repository and publish your site via GitHub Pages. Get one free at github.com/settings/tokens. Your token is sent directly to GitHub's API — it is not stored anywhere.
Is it really free?
Yes, completely free. No credit card required, no subscription, no hidden costs. We use GitHub's free Models API for GPT-4o and GitHub Pages for free hosting. The only thing you need is a free GitHub account.
Can I customize the generated site?
Yes. After the site is built, you can tell the AI what to change using the refinement box. Change colors, layout, content, add sections — the AI will modify the HTML and you can publish the updated version with one click.
