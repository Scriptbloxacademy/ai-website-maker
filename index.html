Frequently asked questions

Everything you need to know about AI Website Maker

What is AI Website Maker?
AI Website Maker is a free tool that uses GPT-4o to build complete professional websites from a description. You describe what you want, and it plans the architecture, generates content, builds the HTML, and publishes it live on GitHub Pages — all in under 2 minutes.
How does it work?
Describe your website in plain English. Pick a template type and style. Our AI runs a multi-step pipeline: first it plans the site architecture, then generates professional content, then builds the HTML using hand-crafted templates. Finally, it creates a GitHub repository and publishes the site via GitHub Pages. You can then refine the site by telling the AI what to change.
Do I need coding experience?
No. Just describe what you want in plain English. The AI handles all the technical implementation — HTML, CSS, responsive design, animations. You don't need to write a single line of code.
What do I need a GitHub token for?
A GitHub classic token with the repo scope is required so the tool can create a repository and publish your site via GitHub Pages. Get one free at github.com/settings/tokens. Your token is sent directly to GitHub's API — it is not stored anywhere.
Is it really free?
Yes, completely free. No credit card required, no subscription, no hidden costs. We use GitHub's free Models API for GPT-4o and GitHub Pages for free hosting. The only thing you need is a free GitHub account.
Can I customize the generated site?
Yes. After the site is built, you can tell the AI what to change using the refinement box. Change colors, layout, content, add sections — the AI will modify the HTML and you can publish the updated version with one click.
How many template types and styles are there?
9 template types: Portfolio, Landing Page, Business, Blog, Store, Web App, Restaurant, Event, and Personal/CV. 9 design styles: Modern, Minimal, Bold, Elegant, Playful, Dark, Glassmorphism, Neumorphism, and Cyberpunk. Any type works with any style.
Can I make a multi-page website?
Yes. Check the "Multi-page" option before building and your site will include Home, About, and Contact pages with navigation between them. Each page is a separate HTML file with consistent design.
Can I add analytics or a contact form?
Yes. Expand the SEO settings below the textarea and check the options for Plausible analytics (free, privacy-friendly) or a contact form. Enter your domain or email and the AI will inject the right code into your generated site.
Can I use my own custom domain?
Yes. After publishing, go to your repo Settings > Pages and enter your custom domain. Add a CNAME record at your domain provider pointing to yourusername.github.io. GitHub will handle HTTPS automatically.
What are integration credits?
AI Website Maker has no credit system. Unlike Base44's paid plans that limit you to 25-1,200 message credits per month, you get unlimited builds with GPT-4o for free. No caps, no meter, no billing.
How does this compare to Base44?
Base44 is a full-stack app platform with auth, databases, and APIs — it costs $20-160/mo. AI Website Maker generates static HTML sites and publishes them free on GitHub Pages. For websites, portfolios, landing pages, and blogs, we're better and free. For apps requiring backend logic (user accounts, data storage), Base44 has advantages.
Do you offer enterprise plans?
Every plan is free. Organizations and enterprises use the same tool — there are no paid tiers. For advanced needs, you can customize generated sites with full code access since the source is in your own GitHub repo.
Can I collaborate with my team?
Yes. Since your site lives in a GitHub repository, you can add collaborators, use pull requests for changes, and manage permissions through GitHub's built-in team tools.
What happens to my data?
Nothing. This tool runs entirely in your browser. Your GitHub token is sent directly to GitHub's API. Your prompt is sent to GitHub's Models API. No data is stored on any server — the entire tool is a single HTML file.
What security measures are in place?
Generated sites use HTTPS (GitHub Pages). Static architecture eliminates SQL injection, XSS, and server-side attacks. Your token is never stored. All changes are git-tracked for full audit trail.
Can I build an API with this tool?
This tool generates static front-end sites only. For APIs and backend functionality, you'd need a separate service. However, you can integrate third-party APIs (like Formsubmit.co for forms or Plausible for analytics) into your generated site.
