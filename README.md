# AI Website Maker

Describe any website in plain English, get a live published link. Free forever.

**How it works:**
1. Type what kind of website you want
2. AI generates the HTML
3. Auto-publishes to GitHub Pages
4. You get a live URL

**Stack:** Gemini API (free) + GitHub Pages (free)

## Hosted version

`https://scriptbloxacademy.github.io/ai-website-maker/`

## Run locally

Just open `index.html` in any browser.
